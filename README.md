# ScrollGuard – Digital Wellbeing App

Break scrolling addiction. Set limits. Track progress. Improve your life.

## Features
- Google Firebase login
- List all installed apps with icons
- Per-app daily limits + custom time windows
- Two modes: Gentle notification OR Strict block (full overlay)
- Real-time usage tracking via Accessibility Service
- Beautiful fl_chart graphs (Daily + Weekly comparison)
- Smart insights
- Clean animations & Material 3 UI
- Hive local storage (offline-first) + Firebase profile
- Bottom navigation + Drawer

## Tech Stack
- Flutter + Riverpod (MVVM)
- Hive + Firebase
- installed_apps + usage_stats + flutter_accessibility_service + fl_chart

## Setup
1. `flutter pub get`
2. Setup Firebase (Google auth)
3. Android: Add Accessibility service in Manifest + xml
4. Grant Usage Stats & Accessibility permissions on first launch
